# Running With Rifles — External Hack

## Features

| Key | Feature | Description |
|-----|---------|-------------|
| **Home** | Fog of War | Disables FOW permanently. Auto-applies at launch. Press to re-apply if needed. |
| **Insert** | Super Camera | Toggles zoomed-out camera for better map awareness. |
| **Page Up / Page Down** | Camera Height | Adjusts camera height (only when super camera is active). |
| **Right Click (hold)** | Triggerbot | Auto-shoots when crosshair is over an enemy. |
| **Left Alt (hold)** | Teleport | Jumps toward crosshair position (max ~3 units per press). Hold to chain jumps. |
| **F1** | Debug Logs | Toggles detailed console output. |
| **End** | Exit | Closes the hack cleanly. |

## Build

Open `running_with_rifles_hack.sln` in Visual Studio 2022, select **Release Win32**, and build.
